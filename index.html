RNA-Seq Analysis App

This app allows the user to input a counts.csv file, perform basic differential gene expression analysis with DESeq2, generate quality control plots, and output results to csv files and plots to a PowerPoint file.

ATAC-Seq Analysis App

This is a Shiny App built to make ATAC-Seq data analysis easier for non-programmers. The app includes data QC, differential peak calling, visualization and downstream annotation steps, all wrapped into a simple point-and-click interface.
