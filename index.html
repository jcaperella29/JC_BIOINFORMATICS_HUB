Single-cell RNA-Seq Solution

Seurat-based workflow for end-to-end single-cell analysis with clear, reproducible steps.

What it does

  • Cell-level QC & filtering (e.g., features/cell, mito/ribo % thresholds)
  • Normalization ‚Üí variable features ‚Üí scaling (with optional covariate regression)
  • Dimensionality reduction: PCA ‚Üí neighbors ‚Üí UMAP/t-SNE
  • Clustering (graph-based) & marker discovery
  • Basic annotation helpers and exportable figures/results

Quick start

  1. Clone the repo:
    git clone https://github.com/jcaperella29/JCAP_SINGLE_CELL_RNA-Seq-Solution
  2. Install dependencies (see repo README for R/Seurat requirements).
  3. Provide a counts matrix (e.g., 10x) or Seurat-ready input.
  4. Run the workflow / launch the UI per the README instructions.

Tip: keep a sample .rds to validate the pipeline quickly.
